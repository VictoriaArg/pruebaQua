# pruebaQua

Antes de comenzar:

- Clonar el repositorio (git clone)
- Instalar las dependencias tanto en la carpeta client como en la carpeta server (npm i)

Para evaluar:

- Desde el directorio server iniciar el servidor con el comando npm start
- Desde el directorio client iniciar el cliente con el comando npm start

Para este challenge mi interpretación del primer ejercicio, cuando se mencionan los 20
archivos .JSON, decidí que si bien la data estaría hardcodeada, la lectura iba a ser
realizada por node.js.
El servidor esta hecho con Express, y el front con React. Utilice hooks y componentes funcionales. Los estilos están hechos con css puro.

Gracias por la oportunidad de demostrar mis conocimientos. Espero seguir en contacto con el equipo para seguir adelante con el proceso de selección.

Saludos!
